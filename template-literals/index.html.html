Template Literals or Template Strings

  1. concatenation two strings

    Old way

    console.log("My Name is Pradeep." + "My age is 26");

    //By template literal

    console.log(`My Name is Pradeep. My age is 26`);

  2. Variable use with strings

    Old way

    let firstName = "Pradeep";

    let lastName = "Pant";

    console.log("My name is " + firstName + ". My Lase name is " + lastName + ".");

    //By template literal

    console.log(`My name is ${firstName}. My last name is ${lastName}.`);

  3. We can add expressions too

    Old way

    let a = 20;

    let b = 5;

    console.log("Total is " + (a + b)) \n "asdasd";

    //By template literal

    console.log(`Total is ${a + b}`);

  4. New Line

    console.log("This is first line \nthis is second line");

    console.log(`This is first line this is second line`);

You can check JS and console for the results
